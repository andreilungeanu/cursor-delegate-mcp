# Privacy

Cursor ACP Bridge runs locally on your machine. It does not send your code,
prompts, or project data to the plugin author.

When you use delegation, prompts and workspace access are handled by
cursor-agent (Cursor) under [Cursor's terms and privacy policy](https://cursor.com/terms).

On first use, the plugin may run `npm install` to fetch open-source
dependencies from the npm registry.

For security issues, see [SECURITY.md](SECURITY.md) or open a
[GitHub Security Advisory](https://github.com/andreilungeanu/cursor-acp-bridge/security/advisories/new).
